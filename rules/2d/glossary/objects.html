Objects

If something isn't a creature, it's probably an object. Objects are ignored by most effects, but when the game statistics are needed for an object that doesn't already have them, use the following rules:
  • An object's hit point maximum and AC are determined by its size, as shown in the Object Stats table.
  • An object's damage resistances, vulnerabilities, immunities, and reductions are determined by its material. See the Materials table.
  • If an object is forced to make a saving throw, it automatically fails.
